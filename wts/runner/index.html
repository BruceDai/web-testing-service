Crosswalk Tests Runner

Running Tests

This tests runner runs some set of Crosswalk tests, current tests version is N/A. The set of online tests available to run can be found in the test repository.

The set of tests run is configurable via the spec name inputting or selection. Some examples of spec name values for input area are:

  • cors - runs all of the cors tests
  • cors, canvas, webaudio: run all tests of cors, canvas and webaudio

The set of online UseCase tests is available to run, which set gather samples to demostrate crosswalk features.
